# Hide It in Your Heart — TMS Verse Quiz

A Bible memory quiz app for the Navigators Topical Memory System (NIV 1984): the 5 Assurances plus TMS Packs A–E (65 verses).

- **Review mode**: type the first letter of each word (title, then verse) to reveal the card, with hints.
- **Test mode**: write the title and verse from memory; word-by-word checking against the card.

## Run locally

```bash
npm install
npm run dev
```

## Deploy to Vercel

1. Push this folder to a GitHub repository.
2. On vercel.com, click **Add New Project** and import the repo.
3. Vercel auto-detects Vite — click **Deploy**.

Or with the CLI: `npm i -g vercel && vercel`
